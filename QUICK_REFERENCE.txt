─────────────────────────────────────────────────────────────────────────────────
ASSESSMENT DIMENSIONS
─────────────────────────────────────────────────────────────────────────────────

Each skill.md is scored 1-10 on:

1. STRUCTURAL COMPLETENESS
   → Does it have all required sections?
   → Frontmatter, overview, examples, gaps documented?

2. THREAT COVERAGE
   → Addresses current attacks (2025-2026)?
   → Maps to OWASP, CWE, industry frameworks?

3. PRACTICAL ACTIONABILITY
   → Can engineers implement it?
   → Code examples, step-by-step guidance present?

4. TOOL & FRAMEWORK CURRENCY
   → References current tools & versions?
   → Aligns with standards (NIST, STRIDE)?

5. CROSS-SKILL INTEGRATION
   → How well does it fit in the broader ecosystem?
   → Avoids duplication with other skills?

Composite Score: Average of 5 dimensions
  • 40-50 = Excellent
  • 30-40 = Good
  • 20-30 = Needs improvement
  • <20  = Critical gaps

TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────────

PROBLEM: Script won't run
SOLUTION: PowerShell execution policy
  Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process

PROBLEM: "File not found" error
SOLUTION: Ensure prompt file and script in same folder as skills directory

PROBLEM: File too large for Claude
SOLUTION: Assess by domain instead
  Example: .\prepare-skills-assessment.ps1 -SkillsDir "03_security skills files\backend"

PROBLEM: Claude output cuts off
SOLUTION: Ask Claude to continue:
  "The response was cut off. Please continue the report from where you left off."

PROBLEM: Unclear what gaps mean
SOLUTION: See SETUP_AND_USAGE_GUIDE.md for detailed explanations
